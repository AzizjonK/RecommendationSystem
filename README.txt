Hardware requirements:
Processor: i5 3rd generation 
Memory: 8 GB
Hard disk space: 256 GB HDD

Software Requirements:
Operating System: Microsoft Windows XP (Sp2 Or Higher)
Software: Google collab, Python
interface: Command Prompt

Basically any PC that runs Google colab will be enough.

Firstly you need to upload ipynb and datasets file to your google drive. 
As you run the code be sure to mount your drive to by inserting a certain key.
Then, for ratings and movies variables replace the paths where you uploaded the datasets.
After that just run every line of your code and you will get output with zero errors.
